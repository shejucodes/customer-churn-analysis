# 📊 Customer Churn Analysis and Prediction using Python, Pandas, Seaborn, and Machine Learning

---

## 📌 Project Overview

The Customer Churn Analysis and Prediction project is a Python based Data Analytics and Machine Learning project designed to analyze telecom customer data and predict customer churn using Logistic Regression.

This project demonstrates a complete end to end Data Science workflow including Data Cleaning, Exploratory Data Analysis (EDA), Data Visualization, Feature Engineering, Model Training, Model Evaluation, and Business Insight Generation.

The objective of this project is to identify key factors contributing to customer churn and build a predictive model that helps businesses reduce churn and improve customer retention.

This project reflects real world responsibilities of Data Analysts and Data Scientists working with business and customer data.

---

## 🚀 What This Project Offers

• 📥 Automated loading and processing of real world telecom customer dataset  
• 🧹 Data cleaning and preprocessing using Pandas  
• 🔍 Exploratory Data Analysis (EDA) to identify churn patterns  
• 📊 Data visualization using Seaborn and Matplotlib  
• ⚙️ Feature engineering using One Hot Encoding  
• 🤖 Machine Learning model training using Logistic Regression  
• 📈 Model performance evaluation using accuracy score, classification report, and confusion matrix  
• 🎯 Identification of key churn drivers and retention factors  

This project demonstrates practical implementation of Data Analytics and Machine Learning workflows.

## ⚙️ How the Project Works

### Step 1: Data Loading
The telecom customer dataset is loaded using Pandas.

### Step 2: Data Cleaning
Missing values are handled, unnecessary columns are removed, and target variable is converted into numerical format.

### Step 3: Exploratory Data Analysis (EDA)
Churn distribution and customer behavior patterns are analyzed using visualizations.

### Step 4: Data Preprocessing
Categorical variables are converted into numerical form using One Hot Encoding.

### Step 5: Model Training
Logistic Regression model is trained using Scikit-learn.

### Step 6: Model Evaluation
Model performance is evaluated using:

• Accuracy Score  
• Classification Report  
• Confusion Matrix  

### Step 7: Feature Importance Analysis
Key churn drivers and retention factors are identified.

---

## 📊 Model Performance

• Model Used: Logistic Regression  
• Accuracy Achieved: 79%  
• Evaluation Metrics Used:
  - Accuracy Score
  - Classification Report
  - Confusion Matrix  

Key Insights:

• Top Churn Driver: Fiber Optic Internet Service  
• Top Retention Driver: Two Year Contract
